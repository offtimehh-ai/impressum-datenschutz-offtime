# Offtime Hamburg - Webseite für Impressum und Datenschutz

Eine einfache, moderne Webseite für Offtime Hamburg mit Impressum, Datenschutzerklärung und einer Übersicht der Community-Mission.

## 📋 Inhalt

- **index.html** - Startseite mit Überblick über Offtime Hamburg
- **impressum.html** - Impressum mit Kontaktdaten und rechtlichen Informationen
- **datenschutz.html** - Datenschutzerklärung gemäß DSGVO
- **style.css** - Responsive Design für alle Geräte

## 🎨 Features

- ✨ Modernes, ansprechendes Design
- 📱 Vollständig responsiv (Mobile, Tablet, Desktop)
- 🏄 Saubere Navigation zwischen den Seiten
- 📋 Umfassende Impressum- und Datenschutztexte
- ⚡ Schnelle Ladezeiten
- ♿ Barrierefreie Struktur

## 🚀 Verwendung

1. Klone das Repository oder lade die Dateien herunter
2. Öffne `index.html` im Browser
3. Navigiere über das Menü zu Impressum und Datenschutz
